- **2026-06-13 (Focus-watch — goose catches you bailing mid-task):** Built the "goose notices you're off-task" feature. **Reality check that shaped it:** a PWA *cannot* read OS Screen Time / Digital Wellbeing or see which other app/site you switched to — that needs **native** (iOS Screen Time API / Android `UsageStatsManager`) or a **desktop browser extension** (active-tab URL), neither shippable today. What a web app *can* do: detect **that** you left Goose via the **Page Visibility API**, and optionally ping you with Web Push. Avi chose the **PWA focus-watch (core only)** path. Implemented in `app/index.html`: a `visibilitychange` listener starts a clock only when you're on the chat screen with unfinished tasks; if you're gone longer than `AWAY_TROLL_MS` (12s — low for demos, bump for real use) and return with work left, the goose roasts you on return via a new `kind:'offtask'` call to `/api/nag` (passes `awaySeconds`; intensity scales with time away). Uses `Date.now()` timestamps (accurate even though the OS pauses page JS while backgrounded). Canned `NAG.offtask` fallback added. **Pitch reframe:** "you walked away from your commitment and I caught you," NOT "I see your TikTok" — feels omniscient without surveillance, and stays clear of the to-do-list trap (pure proactive-character = criterion ①).
